Distributing binaries or APKs also requires preserving copyright and license

counsel. This page is not legal advice.

Third-party Go, npm, Gradle, container, and system packages retain their own
licenses. CI enforces dependency license policy and generates SBOMs; release
bundles must also include third-party inventories. Before adding a dependency,
check license compatibility, security maintenance, size, and alternatives.
Never bypass policy only for development convenience.

Documentation and source contributions are submitted under the repository's
same license unless a file explicitly states otherwise.
