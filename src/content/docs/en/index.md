VoiceAsset is being built as an open-source platform for recording, uploading,
transcribing, reviewing, searching, and safely exposing voice assets to agents.

The project is currently in **Phase 0: foundation**. Product downloads and
deployment instructions will appear only after their corresponding features are
implemented and verified.
